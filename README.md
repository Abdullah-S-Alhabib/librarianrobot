## How to run
To run the code, the following in necessary:
1. Run on Eclipse IDE
2. The EMF addon needs to be installed 
> To install EMF on eclipse: Help -> Install new software -> search for EMF modeling and install it.

## Expected output
The output of the program will be in files:
Readable output text file, which is readable by humans.
And data files for graphing purposes.
